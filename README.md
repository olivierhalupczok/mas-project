# Recruitment Manager

Olivier Halupczok, s32138

Final project based on Mini-project 3 (MP3).

## About

Manages recruitment: companies post jobs, candidates apply with their CVs, and recruiters advance each application through its hiring stages (applied, interviewing, hired, rejected).

## Running for the first time

**Requirements:** Qt 6, qmake, C++17 compiler

```bash
qmake Project.pro
make
./RecruitmentManager
```
